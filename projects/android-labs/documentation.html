BBC News App - Documentation

Android Development Course

About

BBC News App is an Android application that allows users to browse and read BBC news articles. Users can search for specific articles, save their favorites for offline reading, and view detailed article information.

Setup Instructions

Prerequisites

  • Android Studio installed on your computer
  • Git installed
  • Android device or emulator (API level 24 or higher)

Installation Steps

  1. Clone the repository: git clone https://github.com/CinthyaSatar/FinalProject.git
  2. Open Android Studio and select "Open an existing Android Studio project"
  3. Navigate to the cloned project directory and click "OK"
  4. Wait for the project to sync and build
  5. Connect your Android device or start an emulator
  6. Click the "Run" button (green play icon) to install and run the app

Features

  • Browse BBC News articles
  • Search functionality
  • Save articles to favorites
  • Detailed article view
  • Offline access to saved articles

Technical Stack

Android SDK Java SQLite XML BBC News API
