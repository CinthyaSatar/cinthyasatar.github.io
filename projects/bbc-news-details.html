BBC News App

School Project - Android Development Course

Project Overview

The BBC News App was developed as part of my Android Development course. It's a mobile application that provides users with access to BBC news articles, featuring a clean interface and essential functionalities for news consumption and management.

Key Features

  • Real-time news updates via BBC News API
  • Article search functionality
  • Favorite articles management
  • Offline article access
  • Detailed article views with sharing capabilities

Technologies Used

Java Android SDK SQLite XML BBC News API
